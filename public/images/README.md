# Brand Images

Place brand images in this directory with the following names:

- `chanel.jpg` - Image for Coco Chanel / Chanel brand
- `prada.jpg` - Image for Miuccia Prada / Prada brand
- `versace.jpg` - Image for Donatella Versace / Versace brand
- `stella-mccartney.jpg` - Image for Stella McCartney brand

## Image Requirements

- Recommended size: 800x600 pixels or larger
- Format: JPG, PNG, or WebP
- The images will be automatically optimized by Next.js

## Using External Images

You can also use external image URLs by updating the `imageUrl` field in `/data/brands.ts` to point to external URLs (e.g., `https://example.com/image.jpg`).

## Placeholder Images

If images are not provided, the website will automatically display elegant placeholder images with the brand's color scheme.
